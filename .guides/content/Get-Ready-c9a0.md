A game can work perfectly and still not be fun to play. Professional game studios have entire teams dedicated to UI — the part of the game players actually see and interact with. As the video shows, good UI presents information so players can **intuitively understand and act on it**.

In this project, you'll act as a designer to turn an existing game into something that feels like *your* game.
